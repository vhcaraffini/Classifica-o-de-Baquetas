# Classificao-de-Baquetas

## Classificação de Baquetas
Este projeto utiliza técnicas de Machine Learning e Processamento Digital de Sinais (DSP) para classificar diferentes tipos de baquetas com base no som produzido pelo impacto. O objetivo é identificar padrões acústicos que diferenciem modelos, materiais ou marcas de baquetas através da análise de áudio.

## Sobre o Projeto
O projeto foi desenvolvido para automatizar a identificação de baquetas, utilizando extração de características de áudio e modelos de classificação. É uma ferramenta útil para entusiastas de áudio, fabricantes de instrumentos ou bateristas interessados em análise acústica.

### Principais Funcionalidades
Processamento de arquivos de áudio (.wav).

Extração de features acústicas (como MFCCs, Spectral Centroid, Zero Crossing Rate, etc.).

Treinamento de modelos de classificação para identificar o tipo de baqueta.

Visualização de dados e métricas de desempenho.

## Tecnologias Utilizadas
O projeto foi construído utilizando as seguintes bibliotecas e ferramentas:

Python 3.x

Librosa: Para processamento e análise de áudio.

Scikit-learn: Para criação e treino dos modelos de Machine Learning.

Pandas & NumPy: Para manipulação de dados e cálculos matemáticos.

Matplotlib & Seaborn: Para visualização de gráficos e matrizes de confusão.
